Chess AI

I coded this chess AI in 12th grade. It uses Minimax with several modifications, including:

  • Alpha-beta pruning (reduces the number of nodes we have to search)
  • Killer moves (prioritizes moves in which pieces are taken)
  • Iterative deepening (stops evaluating after a certain time limit)
  • Advanced evaluation functions (gives an estimate for who is winning and by how much given a board state)

I'm not very good at chess myself (1100 Lichess rating), so my program was actually able to consistently beat me!
